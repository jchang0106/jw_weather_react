# jw_weather_react

## 앱 소개 및 설명 
- Expo.io와 React를 이용한 날씨 모바일앱 및 반응형 웹앱
- 간단한 Loading Page 구현
- Expo의 Location API와 openWeather를 이용하여 사용자의 GPS위치 파악 및 위도,경도를 통한 실제 현재 기상 데이터로 구현
- Expo API를 통해 얻은 Icon과 Gradient를 통해 디자인

## 앱 살펴보기
- 간단한 로딩페이지
- 현위치, 날씨(아이콘으로 표시), 온도, 날씨 표현으로 구성.
